# yum -y install net-snmp net-snmp-devel net-snmp-utils net-snmp-perl

Available Net-SNMP packages

1. net-snmp    The SNMP Agent Daemon and documentation. This package is required for exporting performance data.

2. net-snmp-libs   The netsnmp library and the bundled management information bases (MIBs). This package is required for exporting performance data.

3. net-snmp-utils  SNMP clients such as snmpget and snmpwalk. This package is required in order to query a system's performance data over SNMP.

4. net-snmp-perl   The mib2c utility and the NetSNMP Perl module.

5. net-snmp-python An SNMP client library for Python.
