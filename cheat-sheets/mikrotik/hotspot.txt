Default settings
/ip address add address=192.168.0.10/24 interface=ether1

Configure Upstream DNS Server:
/ip dns set servers=8.8.8.8 allow-remote-requests=yes

Configure Default Route:
/ip route add dst-address=0.0.0.0/0 gateway 192.168.0.1

Configure interfaces

 Create Loopback Bridge
-  it is used simply as an interface to run the RADIUS server on
/interface bridge add name=Loopback

Add Loopback Bridge IP Address

I use any unused private ip range for this, it is used simply as an interface to run the RADIUS server on.
/ip address add address=10.10.10.1/32 interface=Loopback

Create Hotspot Bridge
/interface bridge add name=Hotspot

Add Hotspot Bridge IP Address
/ip address add address=192.168.5.1/24 interface=Hotspot

Add Hotspot Ports to Bridge
/interface bridge port add bridge=Hotspot interface=wlan1
